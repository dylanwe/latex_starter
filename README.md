# LaTeX starter
A quick starter with some examples for LaTeX.

Compile after changes:
```bash
pdflatex doc.tex
```

Compile for Table of Contents:
```bash
pdflatex doc.tex
pdflatex doc.tex
```

Compile for Bibliography:
```bash
pdflatex doc.tex
bibtex doc
pdflatex doc.tex
pdflatex doc.tex
```
